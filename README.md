# dotfiles
Personal dotfiles.

## Programs
- Distro: Arch Linux
- Window Manager: i3 (i3-gaps)
- Compositor: Picom
- Bar: Polybar
- Lock: i3lock
- Terminal: Alacritty
- Font: Dejavu Sans Mono
- Runner: Rofi
- Text editor: Vim
- Fetch: Neofetch
